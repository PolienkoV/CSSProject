column-gap

Задаёт расстояние между колонками в многоколоночном тексте.

Синтаксис

column-gap: <значение> | normal

Значения

<значение>
Значение ширины межколоночного интервала в единицах CSS (например, пикселах).
normal
Межколоночный интервал устанавливается браузером.

Пример

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Ullam maiores quae incidunt debitis accusamus, quibusdam quasi, obcaecati perferendis commodi neque, animi asperiores quos minus totam quod tenetur eum harum sed.
